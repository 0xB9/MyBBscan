# MyBBscan
Scans `/inc/plugins/` directory for known vulnerable plugins in MyBB 1.8.

## Requirements:
- [Python 2.7](https://www.python.org/downloads/)
- [Huepy](https://github.com/s0md3v/hue) `pip install huepy`
